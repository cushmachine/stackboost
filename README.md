# StackBoost

Landing page for [stackboost.opportune.to](https://stackboost.opportune.to).

StackBoost connects genuine dev tool fans with the companies whose products they love — turning authentic enthusiasm into paid LinkedIn advocacy.

## Deploy

```bash
npx wrangler pages deploy . --project-name=stackboost
```

Then add a CNAME at Namecheap:
- Host: `stackboost`
- Value: `stackboost.pages.dev`
